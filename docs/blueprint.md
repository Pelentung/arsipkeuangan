# **App Name**: Contract Vault

## Core Features:

- Contract Storage: Securely store contract documents in Firestore.
- Contract Metadata: Capture key contract details (e.g., parties, dates, terms) as structured data.
- Search and Filter: Enable users to quickly find contracts based on keywords and metadata filters.
- Reminders & Notifications: Set reminders for important dates and trigger notifications (e.g., upcoming renewals).
- Contract Summarization (AI): Use AI to generate concise summaries of contract terms, helping users quickly understand key provisions; the summarizer can be a tool, extracting only relevant parts of the original.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and security.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#009688) for highlighting key actions and elements.
- Body and headline font: 'Inter' sans-serif font, known for its readability and clean design.
- Use simple, professional icons for contract types, actions, and metadata fields.
- Design a clean, intuitive layout with clear hierarchy and easy navigation.
- Incorporate subtle animations to enhance user experience and provide visual feedback.
